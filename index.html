Welcome

Welcome to the Hike Framework project. The aim of this project is to provide an easy to use 2D Game Engine Framework for Windows Universal App Platform using the C# programming language.

It will use Win2d for graphics to boost the development but keeping in mind an easy way to extend for other libraries like SharpDX or OpenTK.

Projects

For testing, I will use a recent Game Jam prototype and improve it on top of this new framework. Raw project can be found at https://github.com/xesf/gamejam201601
